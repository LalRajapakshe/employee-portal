# Payroll Read Integration Notes

Replace `Payroll.dbo.EmployeeMaster` with the actual payroll source object.

Expected fields:
- EmployeeCode
- FullName
- DepartmentName
- DesignationName
- JoinDate
- EmploymentStatus
- EmploymentType or Permanent flag
- OfficialEmail
- ReportingManagerCode
- DirectorCode
